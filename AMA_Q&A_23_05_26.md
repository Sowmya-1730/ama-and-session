## 1. What is the purpose of DOCTYPE in HTML?
DOCTYPE declares the HTML version to the browser.
It ensures the browser renders the page in **standards mode** instead of quirks mode.

```html
<!DOCTYPE html>
```

## 2. What is the use of OFFSET in SQL?

OFFSET is used to skip a number of rows in query results.
It is mainly used for pagination with LIMIT.
```psql
SELECT * FROM employees
ORDER BY id
LIMIT 10 OFFSET 5;
```

## 3. Different normalization types in SQL

Normalization is divided into forms:
```text
1NF → Atomic values, no repeating groups
2NF → No partial dependency
3NF → No transitive dependency
BCNF → Stronger version of 3NF
4NF → Multi-valued dependency removed
5NF → Join dependency removed
```

## 10. Rules of custom decorators in Python
A decorator is a function that takes another function
It returns a wrapper function
Uses @decorator syntax
```python
def decorator(func):
    def wrapper():
        print("Before function")
        func()
        print("After function")
    return wrapper

@decorator
def hello():
    print("Hello")
```

## 13. Insert duplicate row when primary key exists

Primary key must be unique, so duplicate insert is not allowed.

Solutions:

Change primary key value
Use conflict handling
```psql
INSERT INTO users(id, name)
VALUES (1, 'John')
ON CONFLICT (id) DO NOTHING;
```

## 15. ORDER BY, ASC, DESC and ordering by functions
ORDER BY → sorts results
ASC → ascending order
DESC → descending order
```psql
SELECT * FROM employees
ORDER BY salary DESC;
```
Yes, functions can also be used:
```psql
SELECT name
FROM employees
ORDER BY LENGTH(name);
```
